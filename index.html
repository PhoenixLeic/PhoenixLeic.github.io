The Idle Index is an online project offering free screensavers created by artists, animators and filmmakers based in, or with links to, Leicester. The most recent set of screensavers was commissioned by Phoenix for Leicester Art Week 2020, following on from the editions in 2018, 2019.

We have created an open source version of this extension which can be used by artists and screensaver makers to publish their own screensavers.
