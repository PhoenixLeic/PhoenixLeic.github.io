At Phoenix we are committed to protecting your personal information and being transparent about what we do with it. Our Privacy Policy explains how we use any personal information we collect about you.

The Idle Index Privacy Policy

The Idle Index extension collects data locally, to determine when a user has been idle and the screensaver should start. This data resets each time the screensaver is triggered or an action is taken.

We don't collect any data about how the extension is used.

This extension uses the Idle and Storage permissions API.

We use the storage permissions to store the user screensaver and timing preference.

The Idle Index uses Chrome (Chromuim) or WebExtensions Storage Sync API for storing user's settings and browser's Web Storage API (localStorage) for storing user's screensaver preferences.

This data is all collected locally and Phoenix do not collect or store your data.
